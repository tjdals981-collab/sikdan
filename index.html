👩‍❤️‍👨 부부 공유 설정

가계부 만들 때 쓰던 Firebase 설정을 그대로 붙여넣으면, 두 사람 기기가 같은 식단·메뉴를 실시간으로 함께 봐요. (같은 Firebase 프로젝트를 써도 가계부와 데이터는 섞이지 않아요.)

Firebase 콘솔 → 프로젝트 설정 → '내 앱'의 SDK 설정에 있는 firebaseConfig 부분을 통째로 복사해 붙여넣으면 돼요.

연결되어 있어요. 식단을 바꾸면 두 사람 기기에 바로 반영돼요. 같은 설정을 유진님 기기에도 한 번 넣어두면 끝이에요.
